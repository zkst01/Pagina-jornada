Acerca de la programacion

    Aprender programación y JavaScript requiere paciencia y práctica. Programar puede ser complicado y frustrante, pero cada error es una oportunidad para aprender. Un programa es una construcción de ideas que puede volverse complejo si no se controla. Aunque existen técnicas y mejores prácticas, la creatividad y el aprendizaje constante son esenciales para mejorar como programador.

¿Por qué el lenguaje importa?

Al comienzo de la informática, no existían lenguajes de programación como los conocemos hoy. Los programadores debían escribir instrucciones directamente en código binario (unos y ceros), lo cual era difícil de entender y muy propenso a errores.

00110001 00000000 00000000
00110001 00000001 00000001
...

Para realizar tareas simples, como sumar números, había que manejar instrucciones muy técnicas y poco intuitivas.

Con el tiempo, se comenzaron a usar nombres y símbolos en lugar de números, lo que permitió representar las acciones de forma más clara. Así surgieron los primeros lenguajes de bajo nivel, donde ya se podían entender mejor las operaciones que realizaba un programa. Como por ejemplo el siguiente codigo:

 Establecer "total" como 0.
Establecer "cuenta" como 1.
[loop]
Establecer "comparar" como "cuenta".
Restar 11 de "comparar".
Si "comparar" es cero, ir a [fin].
Agregar "cuenta" a "total".
Agregar 1 a "cuenta".
Ir a [loop].
[fin]
Imprimir "total"

Luego, aparecieron los lenguajes de alto nivel (como JavaScript), que simplifican aún más la escritura de programas. Estos lenguajes permiten usar estructuras como bucles (while) y funciones (console.log) que hacen el código más fácil de leer, escribir y mantener.

let total = 0, cuenta = 1;
while (cuenta <= 10) {
total += cuenta;
cuenta += 1;
}
console.log(total);
// → 55

Incluso, con herramientas modernas, una operación compleja se puede reducir a una sola línea clara y comprensible.

console.log(suma(rango(1, 10)));
// → 55

¿Qué es JavaScript?

JavaScript es un lenguaje de programación que se usa principalmente para hacer páginas web interactivas. Por ejemplo, cuando ves menús desplegables, animaciones, formularios dinámicos o juegos en una página web, probablemente están hechos con JavaScript.

A pesar del nombre, JavaScript y Java no tienen casi nada que ver. El nombre “JavaScript” fue un truco de marketing.

¿Que es ECMAScript?

Cuando JavaScript empezó a usarse en diferentes navegadores, fue necesario crear un estándar para que todos funcionaran de forma compatible.
Así nació ECMAScript, una especificación técnica creada por Ecma International.
ECMAScript es el estándar y JavaScript es su implementación más famosa. Para la mayoría de los desarrolladores, JavaScript y ECMAScript son prácticamente lo mismo.
Evolución del lenguaje

ECMAScript 3 (2000-2010): la versión más usada en los inicios.¿
Versión 4: intentó revolucionar el lenguaje, pero fue cancelada en 2008.
Versión 5 (2009): más simple, pero estable.
2015: llega una gran actualización (la famosa ES6) que cambió todo.
Desde entonces, salen mejoras pequeñas cada año.

Los navegadores modernos se van actualizando para seguirle el ritmo, y los programadores intentan que el código viejo siga funcionando. En este texto se usa JavaScript versión 2017 (ES8).

JavaScript ya no vive solo en las páginas web.
También se usa en:
Bases de datos como MongoDB y CouchDB,
Y en servidores y apps de escritorio gracias a Node.js.

En resumen, JavaScript empezó como un lenguaje para hacer que las webs se movieran… y terminó conquistando el mundo.

Lenguajes más usados y modernos 
  • Python
Muy popular y fácil de aprender. Se usa en inteligencia artificial, ciencia de datos, web, automatización, y casi todo.
Ideal para principiantes.
  • JavaScript
Lenguaje del navegador. Hace que las páginas web sean interactivas. También se usa en servidores con Node.js.

Lenguaje muy estable y usado en empresas, Android y servidores.
“Escribe una vez, corre en todos lados.”
  • C
Uno de los más antiguos aún usados. Es rápido y base de muchos otros lenguajes.
Perfecto para entender cómo piensan las computadoras.

Versión mejorada de C con objetos. Se usa en videojuegos, programas de alto rendimiento y motores gráficos.
Potente, pero complicado.
  • C#
Creado por Microsoft. Se usa para apps, videojuegos (con Unity) y programas de Windows.
Un equilibrio entre potencia y facilidad.
  • PHP
Muy usado en sitios web (como WordPress).
Muchos lo critican, pero sigue alimentando buena parte de Internet.
  • TypeScript
Versión mejorada de JavaScript con tipos. Hace los programas más seguros y organizados.
El favorito de los desarrolladores web serios.
  • Lua
Pequeño y ligero, muy usado en videojuegos para scripts.
Fácil de integrar en otros programas.

    Bucles

    Los bucles permiten ejecutar un bloque de código varias veces. Aquí hay un ejemplo de un bucle for:

    for (let i = 0; i < 5; i++) {
        console.log(i);
}

    Condicionales

    Las estructuras condicionales permiten ejecutar código basado en condiciones. Aquí hay un ejemplo:

    if (x > 10) {
        console.log("x es mayor que 10");
    } else {
        console.log("x es 10 o menor");
    }

    Tipo de Dato Booleano

    Los booleanos son valores que pueden ser true o false. Aquí hay un ejemplo:

    let esVerdadero = true;
    if (esVerdadero) {
        console.log("Es verdadero");
    }

    Funciones

    Las funciones son bloques de código reutilizables. Aquí hay un ejemplo:

    function saludar(nombre) {
        console.log("Hola, " + nombre + "!");
                }

    Operadores

    Los operadores se utilizan para realizar operaciones en variables y valores. Aquí hay un ejemplo:

    const a = 10;
const b = 5;
const suma = a + b;
const resta = a - b;
const multiplicacion = a * b;
const division = a / b;

    Arrays

    Los arrays son estructuras de datos que almacenan múltiples valores en una sola variable. Aquí hay un ejemplo:

    let frutas = ["manzana", "banana", "cereza"];
for (let i = 0; i < frutas.length; i++) {
console.log(frutas[i]);
}
